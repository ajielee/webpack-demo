# webpack-demo

This is a very simple webpack demo for my presentation.

## Requirement

- nodejs
- yarn

## How to play with this demo

1. Install dependencies

  ```bash
  yarn install
  ```

2. Compile `app.js`

  ```bash
  npm run build
  ```

3. Use `webpack-dev-server` to run the devServer

  ```bash
  npm run serve
  ```

4. Show it on your navigator: `http://localhost:8888`
